# Topics Covered

## 1. Introduction to Full Stack Development

Full Stack Development includes:

* Frontend Development
* Backend Development
* Database Management

### Frontend

Frontend is the visible part of a website that users interact with.

Technologies:

* HTML
* CSS
* JavaScript
* React

### Backend

Backend handles server-side logic and data processing.

Technologies:

* Node.js
* Express.js

### Database

Database stores website/application data.

Examples:

* MongoDB
* MySQL

# 3. Introduction to HTML

HTML stands for HyperText Markup Language.

HTML is used to create the structure of webpages.

---

# Basic HTML Structure

```html
<!DOCTYPE html>
<html>
<head>
    <title>My Website</title>
</head>
<body>

</body>
</html>
```

---

# HTML Tags Learned

## Heading Tags

```html
<h1>Heading</h1>
<h2>Heading</h2>
```

## Paragraph Tag

```html
<p>This is paragraph.</p>
```

## Button Tag

```html
<button>Click Me</button>
```

## List Tag

```html
<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>
```
